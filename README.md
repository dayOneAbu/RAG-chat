# RAG Chat - AI Chatbot with Retrieval-Augmented Generation

AI-powered chatbot that answers questions based on your documents using RAG technology.

## Features
- Document upload and embedding generation
- Semantic search with vector database
- OpenAI GPT integration for natural responses
- Chat history and context management
- TypeScript for type safety
